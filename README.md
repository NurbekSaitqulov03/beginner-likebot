# beginner-likebot

## objective

This is a simple bot that will count the number of likes and dislikes on telegram messages.

## technical requirements

- when press the start commmand, the bot will send welcome message with two buttons, one for like and one for dislike
- when press the button, the bot will send a message with the number of likes and the number of dislikes

## setup

1. Create a virtual environment

    ```bash
    python3 -m venv venv
    ```

2. Activate the virtual environment

    linux and mac

    ```bash
    source venv/bin/activate
    ```

    windows

    ```bash
    source venv/Scripts/activate
    ```

3. Install the requirements

    ```bash
    pip install -r requirements.txt
    ```
